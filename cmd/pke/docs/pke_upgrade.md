## pke upgrade

Upgrade a single Banzai Cloud Pipeline Kubernetes Engine (PKE) machine

### Synopsis

Upgrade a single Banzai Cloud Pipeline Kubernetes Engine (PKE) machine

```
pke upgrade [flags]
```

### Options

```
  -h, --help                        help for upgrade
      --kubernetes-version string   Kubernetes version (default "1.14.0")
```
